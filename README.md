## Abstract

Decoding auditory target perception from EEG signals in complex acoustic scenes (e.g., the cocktail party problem) is a key challenge in auditory brain–computer interfaces. This work systematically analyses neural representations underlying auditory attention and derives a brain-inspired deep learning architecture that mirrors the hierarchical processing observed in auditory cortex. The proposed model achieves state-of-the-art accuracy on the CAS-EEG dataset while maintaining low computational cost, making it suitable for real-time assistive applications.
